# AZ-900 クラウド基礎Pass プライバシーポリシー

最終更新日: 2026年7月4日

AZ-900 クラウド基礎Pass（以下「本アプリ」）は、AZ-900 / Microsoft Azure Fundamentals の学習、広告表示、アプリ内課金、品質改善のために必要な範囲で情報を取り扱います。本ポリシーは、本アプリにおける情報の取り扱いについて説明するものです。

## 1. 本アプリが取り扱う情報

### 1.1 端末内に保存される学習データ

本アプリは、以下の学習データを利用者の端末内に保存します。

- 回答履歴
- 正誤履歴
- 苦手問題の情報
- 分野別の学習進捗
- 誤答優先学習のための正答数・誤答数
- 連続学習日数
- 文字サイズなどの設定情報

これらの学習データは、問題演習、復習、学習進捗表示、誤答優先学習、苦手問題モード、連続学習日数の表示のために使用します。現時点で、本アプリ独自のアカウント登録機能やサーバー同期機能はありません。

### 1.2 広告に関する情報

本アプリは広告表示のために Google Mobile Ads SDK（AdMob）を利用する場合があります。広告配信、広告の品質管理、不正防止、効果測定のため、端末情報、広告の表示・操作に関する情報などが Google により取り扱われる場合があります。

無料ユーザー向けに、バナー広告、インタースティシャル広告、リワード広告を表示する場合があります。

バナー広告は、学習画面や設定画面などの下部に表示される場合があります。インタースティシャル広告は、学習完了後など、利用の区切りとなるタイミングで表示される場合があります。

リワード広告は、利用者が「広告視聴で問題を追加解放」などのボタンを明示的に操作した場合にのみ表示を試みます。広告視聴を完了した場合に限り、追加の問題が一時的に解放されます。広告の読み込み失敗、広告在庫なし、タイムアウト、表示失敗、途中終了の場合は、追加解放は行われません。

Google による情報の取り扱いについては、Google のポリシーをご確認ください。

- Google プライバシーポリシー: https://policies.google.com/privacy
- Google 広告に関するポリシー: https://policies.google.com/technologies/ads

### 1.3 アプリ内課金に関する情報

本アプリはアプリ内課金の購入状態を管理するために RevenueCat SDK を利用する場合があります。購入状態の確認、復元、プレミアム機能の有効化のため、購入情報、アプリユーザーID、端末情報などが RevenueCat および Apple により取り扱われる場合があります。

本アプリの開発者は、Apple が管理する決済情報のうち、クレジットカード番号などの支払い手段の詳細を取得しません。

各サービスによる情報の取り扱いについては、以下をご確認ください。

- RevenueCat Privacy Policy: https://www.revenuecat.com/privacy/
- Apple Privacy Policy: https://www.apple.com/legal/privacy/

### 1.4 Firebase Analytics に関する情報

本アプリは利用状況の把握と品質改善のために Firebase Analytics を利用する場合があります。Firebase は Analytics only として利用し、Firebase Authentication、Firestore、Realtime Database、Functions、Storage、Remote Config、Crashlytics、Cloud Messaging、App Check は利用しません。

Firebase Analytics では、アプリ起動、学習開始、回答結果、学習完了、実戦演習開始・完了、購入導線表示、購入開始・完了、レビュー依頼条件到達などのイベントを送信する場合があります。送信する情報は、分野ID、問題数、正解率、正誤、プレミアム状態など、学習機能の改善に必要な範囲に限定します。

問題本文、選択肢本文、解説本文、問題IDの生値、自由入力、氏名、メールアドレス、購入トランザクションID、広告ID、SDKキー、アプリ固有の内部IDなどは Firebase Analytics イベントに送信しません。

本アプリは Firebase Analytics について IDFA を使用しない構成を採用しており、App Tracking Transparency の許諾ダイアログは不要と判断しています。

Firebase による情報の取り扱いについては、Google のポリシーをご確認ください。

- Google プライバシーポリシー: https://policies.google.com/privacy

### 1.5 レビュー依頼に関する情報

本アプリは、一定の学習条件を満たした場合に、Apple が提供するレビュー依頼機能を表示する場合があります。レビューの入力、送信、表示、管理は Apple の仕組みにより行われ、本アプリの開発者がレビュー内容と利用者個人を結び付けて取得することはありません。

### 1.6 クラッシュ・不具合調査に関する情報

本アプリの安定性向上、不具合調査、品質改善のため、クラッシュ情報、OS バージョン、端末種別、アプリバージョンなどの技術情報が Apple、Expo、または利用している開発基盤により取り扱われる場合があります。

### 1.7 お問い合わせ時に利用者から提供される情報

利用者がお問い合わせメールを送信した場合、メール本文に記載された内容、返信先メールアドレス、端末やアプリの状態に関する情報を、問い合わせ対応のために利用します。お問い合わせで取得した情報は、回答、調査、再発防止のために必要な範囲でのみ使用します。

## 2. 利用目的

本アプリは、取り扱う情報を以下の目的で利用します。

- 問題演習、復習、学習進捗表示の提供
- 誤答優先学習、苦手問題モード、連続学習日数表示の提供
- 文字サイズなどの利用者設定の保存
- 広告の表示、広告品質の管理、不正防止
- 利用状況の分析と学習体験の改善
- アプリ内課金の購入状態確認、復元、プレミアム機能の提供
- クラッシュ、不具合、表示崩れなどの調査と改善
- 利用者からのお問い合わせへの対応

## 3. 第三者提供・外部送信

本アプリは、広告配信、アプリ内課金、品質改善のため、以下の外部サービスを利用する場合があります。各サービスで取り扱われる情報は、それぞれのプライバシーポリシーに従って管理されます。

- Google Mobile Ads SDK（AdMob）
- Firebase Analytics
- RevenueCat
- Apple App Store / StoreKit
- Expo および Apple が提供するクラッシュ・診断関連機能

本アプリの学習履歴は、原則として利用者の端末内に保存されます。ただし、OS、SDK、外部サービスの仕様により、広告、購入、分析、クラッシュ、診断に関する情報が各サービスへ送信される場合があります。

## 4. 収集しない情報

本アプリは、以下の情報を本アプリ独自の機能として収集しません。

- 氏名、住所、電話番号
- 連絡先
- 正確な位置情報
- 写真、カメラ、マイクの内容
- 本アプリ独自のログインID、パスワード
- 学習履歴を外部サーバーへ同期するためのアカウント情報
- Microsoft、Azure、試験申込サイト、試験実施団体、関連団体のアカウント情報
- 外部AIサービスのAPIキー

お問い合わせメールを送信した場合は、利用者がメールに記載した情報を問い合わせ対応のために取り扱います。

## 5. 学習データの削除

利用者は、アプリ内の設定画面から学習データをリセットできます。アプリを削除した場合も、端末内に保存された学習データは削除されます。

なお、Apple、Google、RevenueCat など外部サービス側で管理される購入情報、広告関連情報、クラッシュ情報については、各サービスのポリシーおよび設定に従って管理されます。

## 6. 未成年の利用

本アプリは資格学習を目的としたアプリです。未成年の方が利用する場合は、必要に応じて保護者の方と一緒に内容を確認してください。
